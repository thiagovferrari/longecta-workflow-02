# Guia de Deploy na Vercel

Seu projeto está pronto para ir ao ar! Siga estes passos simples:

1. Acesse [vercel.com/new](https://vercel.com/new).
2. Selecione o repositório **longecta-workflow-02**.
3. Na seção "Environment Variables", adicione estas duas chaves (copie do arquivo `.env` local):
   - `VITE_SUPABASE_URL`: ...
   - `VITE_SUPABASE_ANON_KEY`: ...
4. Clique em **Deploy**.

O sistema já está configurado com `vercel.json` para funcionar perfeitamente com rotas do React.
